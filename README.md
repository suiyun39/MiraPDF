# MiraPDF

一个简洁、易用的 PDF 阅读器。

## 功能特性

- 📖 原生技术打造，提供最纯粹的阅读体验。
- 📝 精心打磨的标注功能，简单、但足够好用。
- 🤖 AI 助手，生成摘要、提取结构、整理笔记、语义搜索都不在话下。

## Roadmap

**基础阅读能力**

- [ ] 单页、双页、连续滚动多模式阅读
- [ ] 侧边缩略图、缩图清单视图
- [ ] 目录解析、书签管理、标注管理
- [ ] 全局搜索、标注备注查找

**阅读标注**

- [ ] 文本高亮、批注
- [ ] 文本下划线、删除线、波浪线
- [ ] 矩形、椭圆、折线、箭头、文本、便签
- [ ] PDF 表单

**AI 助手**

- [ ] 摘要、大纲、关系图生成
- [ ] 语义搜索、问答模式、翻译改写
- [ ] 笔记标注整理提取
- [ ] 自动跳转、自动标注

**功能增强**

- [ ] 双语对照阅读
- [ ] OCR 识别
- [ ] PDF 分割、合并、排列
